NP Sprite Sheet

Convert animations to sprite sheets easily!

Usage

Create different animations as separate layers.
Then, press "export" and choose where to save the file.
The plugin will export to where you've chosen and save "[filename].ss.json" along with it.
The JSON file describes each animation for loading into other programs. It contains the following data:
  • anims: an object mapping the names of each animation to objects with these values:
    • offset: the starting frame, going left to right, top to bottom. The frames of the animation are in sequential order.
    • key_times: the number of frames to hold each keyframe until switching to the next.
      Important note: the last frame will only be held for 1 frame, as the size of an animation is determine per document, not per layer, and can't be used to determine the correct value. To hold the final frame longer, add "+X" to the end of the layer's name, replacing "X" with the amount of holding frames to add.
  • framerate: the frames per second as an integer
  • size: the number of rows, then columns
